# zaikon

## スクリプトの起動
> npm run dev

## アイコンたち
| 機能    | テキストボタン           | アイコン（Font Awesome）             |
| ----- | ----------------- | ------------------------------ |
| 追加    | `追加`              | `<i class="fas fa-plus"></i>`  |
| 削除    | `削除`              | `<i class="fas fa-trash"></i>` |
| 変更    | `変更`              | `<i class="fas fa-pen"></i>`   |
| ステータス | `Red`, `Green` など | 色付き丸いアイコン（下記参照）                |
